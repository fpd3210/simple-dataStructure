  - 桥接模式
  - 装饰模式
  - 组合模式
  - 外观模式
  - 享元模式
  - 代理模式
- 行为型模式（11种）
  - 模板方法模式
  - 命令模式
  - 访问者模式
  - 迭代器模式
  - 观察者模式
  - 中介者模式
  - 备忘录模式
  - 解释器模式
  - 状态模式
  - 策略模式
  - 责任链模式
